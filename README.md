# Fake News Generator & Detector using Generative AI

An AI-powered project that detects fake news using Machine Learning and generates synthetic news articles using Generative AI.  
---

##  Features
-  Fake News Detection ‚Äî Classifies news as *real* or *fake* using ML/NLP models.  
-  News Generation ‚Äî Generates realistic fake news using a generative model.  
-  Model Training & Evaluation ‚Äî Includes training, testing, and performance evaluation.  
-  Modular Code ‚Äî Separate modules for generator and detector.

---

##  Tech Stack
- **Language:** Python  
- **Libraries:** TensorFlow / PyTorch, Scikit-learn, Pandas, NumPy  
- **NLP Tools:** NLTK, Hugging Face Transformers  
- **Dataset:** [Fake and Real News Dataset](https://www.kaggle.com/clmentbisaillon/fake-and-real-news-dataset)
